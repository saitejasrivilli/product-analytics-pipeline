Design Decisions

  • Star Schema: Fact + dimensions for query performance. Denormalization acceptable for analytics.
  • Order-Product Grain: One row per order-product enables product-level funnel analysis.
  • Incremental fct_orders: Only processes new orders (order_id > max_id). Reduces runtime from 1.3s → 0.49s on repeat runs.
  • dbt Tests: All models have comprehensive test coverage (not_null, unique, referential integrity).
